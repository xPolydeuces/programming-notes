# C language basics

## Data types in C

| Data Type | Range |
|-----------|-------|
| int | -2^31^ - 2^31^ -1 |
| unsigned int | 0 - 2^32^ -1 |
| char | -128 - 127 |
| float | eg. 3.14 |
| double | like float, can store bigger nums |
| void | type =/= data type |

## CS50 library

| function | return |
|-----------|-------|
| bool | true/false |
| string (char*) | eg. "hello" |
| get_char | char input |
| get_double | double input |
| get_int | int input |
| get_float | float input |
| get_long | long input |
| get_string | string input |
